# Column size properties

This article explains how to setup column sizes.


<br/>


#### Column resize
```js
const grid = div.querySelector('revo-grid');
grid.resize = true;
```


<br/>


#### Column width
```tsx
const columns = [{ name: 'Name', prop: 'name', size: 500 }];
<revo-grid columns={columns}/> // and then in tsx
```
